# shopping-trends-dashboard
An interactive Power BI dashboard analyzing shopping trends, customer purchasing behaviors, item performance, and location-based sales insights.
Shopping Trends Power BI Dashboard

📌 Overview

This project features a Shopping Trends Dashboard built in Power BI. It provides a comprehensive view of retail performance, allowing stakeholders to analyze customer purchasing habits across different product categories, seasons, and geographic locations. The dashboard is highly interactive, featuring detailed breakdowns of top and bottom-performing items to help drive inventory and marketing decisions.

📊 Key Performance Indicators (KPIs)

The dashboard tracks the following primary metrics at a glance:
•	Sum of Purchase Amount (USD): 233K
•	Sum of Previous Purchases: 99K

📈 Visualizations & Insights

The dashboard includes several visualizations and tables to analyze retail data from multiple angles:

•	Total Purchase Amount by Category (Bar Chart): Highlights sales by clothing type, showing Clothing as the top-selling category, followed by Accessories, Footwear, and Outerwear.
•	Total Purchase Amount by Season (Area Chart): Illustrates seasonal spending trends, indicating that Fall drives the highest purchase amounts, while Summer has the lowest.
•	Item Performance Tables: 

o	Top 10 / Bottom 10 Items Purchased: Identifies top sellers (e.g., Blouse, Dress, Shirt) and lower-performing items (e.g., Jeans, Sneakers, Backpack) by purchase amount.
o	Top 10 / Bottom 10 Reviews: Ranks items based on the sum of their review ratings, allowing for quick correlation between sales and customer satisfaction.
•	Location Performance Tables: Details the Top 10 and Bottom 10 Locations based on previous purchases, highlighting states like Montana and Minnesota as top performers.

🎛️ Interactive Filters (Slicers)

To allow for deep-dive analysis, the dashboard includes dynamic filters on the right panel and top header:
•	Shipping Type: 2-Day Shipping, Express, Free Shipping, Next Day Air, Standard, Store Pickup.
•	Demographics & Product Attributes: Dropdowns for Color and Age, along with button selectors for Size (L, M, S, XL).
•	Category: Accessories, Clothing, Footwear, Outerwear.
•	Season: Fall, Spring, Summer, Winter.
•	Payment Method: Bank Transfer, Cash, Credit Card, Debit Card, PayPal, Venmo.

🛠️ Tools Used

•	Power BI: For data visualization and dashboard design.

🚀 How to Use

1.	Clone this repository: git clone https://github.com/your-username/shopping-trends-dashboard.git
2.	Open the .pbix file using Microsoft Power BI Desktop.
3.	Interact with the slicers on the right panel and the shipping options at the top to filter the data dynamically by category, payment method, season, or customer demographics.
